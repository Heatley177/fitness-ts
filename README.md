# React/TypeScript/Vite Fitness App
A responsive fitness web application

# Features
- Single page responsive design
- Mobile hamburger menu
- Automatic, looping carousel
- Sign-up E-mail form and Contact Us page
- Checkout more for yourself [HERE](https://fitness-ts-73c.pages.dev/)
  
# Technology
- Vite
- React
- Typescript
- Tailwind
- Framer Motion
- Hero Icons
- Form Submit
- React Anchor Link Smooth Scroll
- Deployed using Cloudflare Pages
